# FraudShield

FraudShield is a modular fraud detection platform designed to identify suspicious financial transactions using rule-based logic, risk scoring, and AI-assisted insights. The system is structured to reflect real-world fintech architectures.

## Overview

The platform simulates production-style fraud detection systems by combining rule engines, analytics, and backend APIs with a monitoring dashboard.

## Key Features

- Rule-based fraud detection (velocity, geo anomaly, high-value transactions)
- Risk scoring engine with classification
- FastAPI-based backend services
- Analytics dashboard (Streamlit)
- Synthetic data generation for testing
- AI-assisted fraud explanations

## Architecture

- Dashboard (Streamlit)
- API Layer (FastAPI)
- Core Engine (rules, scoring, analytics, AI)
- Database (SQLite)

## Tech Stack

Python, FastAPI, Streamlit, SQLite, Gemini AI

## Highlights

- Modular backend architecture
- Real-world system design approach
- Hybrid rule-based and AI-assisted detection
